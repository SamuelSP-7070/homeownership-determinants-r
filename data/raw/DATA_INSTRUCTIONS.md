# Data Access Instructions

The raw dataset used in this project is derived from the
American Community Survey (ACS) 2022 microdata via IPUMS USA.

Due to file size limitations and data usage restrictions,
the raw dataset is not included in this repository.

## How to Reproduce the Analysis
1. Visit https://usa.ipums.org/usa/
2. Create a free IPUMS account
3. Extract ACS 2022 data with the following variables:
   - INCTOT, FTOTINC
   - AGE, SEX, MARST
   - RACE, EDUC
   - FAMSIZE, NCHILD
   - CLASSWKR
